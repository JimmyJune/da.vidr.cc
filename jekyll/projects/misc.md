### Cube

Just a wireframe [SDL][1] cube. To download: 
    
{% highlight bash %}
git clone git://github.com/davidar/sdl-cube.git
{% endhighlight %}

To build: 
    
{% highlight bash %}
cd sdl-cube && mkdir build && cd build && cmake .. && make && ./cube
{% endhighlight %}

   [1]: http://www.libsdl.org/
